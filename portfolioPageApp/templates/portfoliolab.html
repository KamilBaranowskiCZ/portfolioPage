This is a charity application made as a part of a project at the Coders Lab school. The html, css and javascript files were provided by Coders Lab. The back-end and a part of JS scripts were written by myself. The app serves people who want to support charitable foundations. It has an adminstrator panel where you can add new institutions. On the user's side, there is an interactive form thanks to which we can report a willingness to support a selected foundation. Users has the option to change their data, such as e-mail or name, and to view the collections supported so far.
